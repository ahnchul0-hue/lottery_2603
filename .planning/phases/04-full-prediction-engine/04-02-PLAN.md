Research-provided code pattern (largest-remainder rounding):
```python
def round_to_sum(ratios: list[float], target: int = 6) -> list[int]:
    floors = [int(r) for r in ratios]
    remainders = [r - f for r, f in zip(ratios, floors)]
    deficit = target - sum(floors)
    indices_by_remainder = sorted(range(len(remainders)),
                                   key=lambda i: remainders[i], reverse=True)
    for i in range(deficit):
        floors[indices_by_remainder[i]] += 1
    return floors
```

Research-provided balance category computation:
```python
def compute_category_counts(odd_count, even_count, high_count, low_count) -> dict[str, int]:
    odd_high = min(odd_count, high_count)
    odd_low = odd_count - odd_high
    even_high = high_count - odd_high
    even_low = even_count - even_high
    if even_low < 0:
        adjustment = -even_low
        odd_high -= adjustment
        odd_low += adjustment
        even_high += adjustment
        even_low = 0
    return {"odd_low": odd_low, "odd_high": odd_high, "even_low": even_low, "even_high": even_high}
```
