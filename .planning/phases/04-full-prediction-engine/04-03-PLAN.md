```python
def normalize_scores(scores: dict[int, float]) -> dict[int, float]:
    MIN_FLOOR = 0.001
    floored = {n: max(s, MIN_FLOOR) for n, s in scores.items()}
    total = sum(floored.values())
    return {n: s / total for n, s in floored.items()}
```

    CRITICAL per research: CompositeStrategy does NOT instantiate or call other strategies' generate() methods. It computes its own per-number (1-45) scores using the same data signals and combines them.

    Algorithm per D-07, D-08:

    Step 1 - COMPUTE FREQUENCY SCORES: Use weighted_frequencies directly (already dict[int, float]).

    Step 2 - COMPUTE PATTERN SCORES: For each number n (1-45), compute a pattern affinity score:
    - Pair score: Sum of decay-weighted pair frequencies involving number n (using itertools.combinations on draws, weighted by per-draw decay weights computed from len(draws) and config.DECAY_HALFLIFE).
    - Ending-digit score: Decay-weighted frequency of n's ending digit (n % 10) across all draws.
    - Combine: pair_score + ending_digit_score for each number.

    Step 3 - COMPUTE RANGE SCORES: For each number n, compute expected zone density = (zone's weighted count) / (zone size). This gives each number in the same zone an equal share of that zone's statistical weight.

    Step 4 - COMPUTE BALANCE SCORES: For each number n, compute how often its category (odd/even x high/low) appears in the draws' ratio distributions, weighted by decay. This gives a per-number balance-fitness score.

    Step 5 - NORMALIZE each score set to sum=1.0 using normalize_scores helper (apply MIN_WEIGHT_FLOOR first to prevent zero-probability).

    Step 6 - WEIGHTED AVERAGE per D-07:
    composite_score[n] = 0.40 * freq_norm[n] + 0.20 * pattern_norm[n] + 0.20 * range_norm[n] + 0.20 * balance_norm[n]

    Step 7 - PROBABILISTIC SELECTION: Use composite_scores as weights with random.choices (same as FrequencyStrategy). Apply _select_unique + _generate_diverse_games pattern.
