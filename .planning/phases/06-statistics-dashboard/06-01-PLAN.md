    ```python
    from app.schemas.lottery import LotteryDraw

    def compute_heatmap_data(
        by_machine: dict[str, list[LotteryDraw]],
    ) -> list[dict]:
        machines = ["1호기", "2호기", "3호기"]
        result = []
        for machine in machines:
            draws = by_machine.get(machine, [])
            total = len(draws)
            expected_per_number = total * (6 / 45)

            freq: dict[int, int] = {n: 0 for n in range(1, 46)}
            for draw in draws:
                for num in draw.numbers:
                    freq[num] += 1

            deviations: dict[str, float] = {}
            for num in range(1, 46):
                if expected_per_number > 0:
                    deviations[str(num)] = round(
                        (freq[num] - expected_per_number) / expected_per_number, 4
                    )
                else:
                    deviations[str(num)] = 0.0

            result.append({
                "machine": machine,
                "deviations": deviations,
                "total_draws": total,
            })
        return result
    ```

    **4. Create `frontend/src/hooks/useStatistics.ts`** (per D-04, D-06):

    A single hook computing ALL frontend statistics via useMemo from the draws array. Key computations:

    - **frequencyData**: Initialize counts 1-45 to 0, iterate all draws counting each number. Return as `NumberFrequency[]` sorted by number.
    - **hotNumbers**: Top 10 numbers by frequency count (sort descending, slice 0-10, map to number).
    - **coldNumbers**: Bottom 10 numbers by frequency count (sort ascending, slice 0-10, map to number).
    - **oddEvenDist**: Parse each draw's `odd_even_ratio` string (e.g. "3:3") by splitting on `:`. Count occurrences of each ratio across all draws. All 7 categories: "0:6", "1:5", "2:4", "3:3", "4:2", "5:1", "6:0". Return as `RatioDistribution[]`.
    - **highLowDist**: Same as oddEvenDist but using `high_low_ratio` field. Same 7 categories.
    - **zoneDist**: For each draw's numbers, categorize each number into zones [1-9], [10-19], [20-29], [30-39], [40-45]. Sum across all draws. Calculate percentage as `count / (draws.length * 6) * 100`. Return as `ZoneDistribution[]` with 5 entries.
    - **sumDist**: Bucket each draw's `total_sum` into 20-unit bins: "21-40", "41-60", "61-80", "81-100", "101-120", "121-140", "141-160", "161-180", "181-200", "201-220", "221-240". Count draws per bin. Return as `SumDistribution[]` (only bins with count > 0 needed).
    - **acDist**: Count occurrences of each `ac_value` (range 2-5, only 4 distinct values per research). Return as `AcDistribution[]` sorted by acValue ascending.

    Use cascaded useMemo: frequency depends on draws, hotNumbers/coldNumbers depend on frequency, others depend on draws directly.

    Function signature: `export function useStatistics(draws: LotteryDraw[]): StatisticsResult`

    Import `LotteryDraw` from `'../types/lottery'` and all result types from `'../types/statistics'`. Use `export type` import syntax per verbatimModuleSyntax convention.
