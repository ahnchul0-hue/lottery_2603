       ```python
       class PredictionStrategy(ABC):
           @property
           @abstractmethod
           def name(self) -> str: ...

           @property
           @abstractmethod
           def display_name(self) -> str: ...

           @abstractmethod
           def generate(
               self,
               draws: list[LotteryDraw],
               weighted_frequencies: dict[int, float],
           ) -> list[list[int]]: ...
       ```

       - `generate()` method:
